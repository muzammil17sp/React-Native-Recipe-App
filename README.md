# Installation Steps

## Using npm

Run commands

1) ```npm install```


2) ```npx react-native start```

3) ```npx react-native run-android``` 


## Or using yarn

Run commands 

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn react-native start```

4) ```yarn react-native run-android```
